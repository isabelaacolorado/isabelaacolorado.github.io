NileDotCom Application

NileDotCom is a Java-based electronic store simulation project that demonstrated event-drive GUI programming concepts. This application allows users the ability to browse throug products, add them to a shopping cart, and proceed through a simulated checkout process. It also logs transactions and records the time at which they take place.

Key Features

  • Interactive GUI for product browsing, shopping cart management, and checkout process.
  • Dynamic labeling and price updates in real-time.
  • Transaction logging for order history and time.
  • Error handling and state management
